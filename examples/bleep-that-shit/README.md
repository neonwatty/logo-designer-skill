# Bleep That Sh*t! — Logo Design Iteration Showcase

This example documents the complete logo design process for [Bleep That Sh\*t!](https://bleepthatshit.com), a profanity-bleeping app. The project used the logo-designer-skill to go from a blank canvas to a polished, production-ready logo through 37 iterations across 10 distinct design phases.

The journey started with five divergent concepts (speech bubbles, waveforms, megaphones, pixel blocks, and asterisks), narrowed down to a waveform + censor metaphor, pivoted to a HAL 9000 eye concept, and culminated in a comic book-styled icon with halftone dots, bold outlines, and a signature red-and-yellow palette. Each phase tackled a specific design challenge: readability at favicon sizes, visual personality matching the app's irreverent tone, and fine-tuning proportions for the final polish.

Open [`showcase.html`](showcase.html) in a browser to view the full interactive showcase. Every SVG from the design process is included — click any thumbnail to view it full-size. The page works as a local `file://` URL with no build step or server required.
